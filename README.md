# Melly+

App de registro de glucosa para adultos mayores.

## Características

- Interfaz simple y accesible con letras grandes
- Registro de glucosa en días específicos (Lunes, Miércoles, Viernes)
- Almacenamiento local de datos
- Historial de mediciones
- Feedback visual con colores según niveles

## Tecnologías

- React Native
- Expo
- AsyncStorage
- React Navigation

## Instalación

```bash
npm install
npx expo start
```

## Uso

Escanea el QR con Expo Go en tu dispositivo móvil.
